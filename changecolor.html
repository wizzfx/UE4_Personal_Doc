Press E to Change Color For Button ( Material Dynamic Instance )

First create master material like below.
We will use Green_Red parameter in Material Instance to dynamicaly change colors.

Create Material Instance out of your master.

Create new Blueprint Class Actor called ( BP_Button ). Add Mesh ( Box ) to the blueprint scene.
Assign Material Instance to this Mesh

Go to Construction Script tab and put Button Mesh on it. Add node ( Create Dynamic Variable ).
Press RMB on Return Value and choose ( Promote to Variable ). Change variable name to ( DynamicVar ) and make it Public ( check Instance Editable option ).

Go to Event Graph. Create custom event called ( Button_Color_OnOff ) like on image below.
Scalar Parameter is our Parameter inside Material Instance.
Put Parameter Name and its Value. BP_Button is done!

Create blueptint class actor BP_Monitor. Put simple mesh inside scene ( Box ). You can also create Tooltip ( Text ). And BoxTrigger.
Create Variable ( your variables can have different types. Int, float or even Blueprint ).
In this case your Variable should have TYPE ( BP_Button_OnOff ), that is just blueprint that we've created previously. Make this variable public.

Last thing. Put two blueprint on the scene. Select BP_Monitor and in Details select Default Button State. It should be our BP_Button_OnOff.

Done. Now if you go into BP_Monitor and activate trigger you will be able to use E key to change color of the button inside BP_Button_OnOff.
